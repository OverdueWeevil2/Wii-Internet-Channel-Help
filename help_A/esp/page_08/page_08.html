8      
  Control parental
       
 
 
 
Existen dos métodos para restringir el uso del Canal Internet.
 Mediante el uso del control parental
El control parental es una herramienta de la consola Wii. Los padres y los tutores pueden restringir ciertas funciones que utilizan esta herramienta. El acceso al Canal Internet puede ser restringido completamente al seleccionar el tipo de control parental correspondiente. Para activar el control parental, visita la pantalla del control parental que se encuentra en las opciones de Wii. Para recibir más información, consulta la sección de “Control parental” en el manual de operaciones de la consola Wii.
 Uso de un servicio filtrador de contenido
Puedes usar un servicio de un tercero para filtrar contenido (se vende por separado) que restrinja el acceso a páginas de internet que puedan tener contenido ofensivo. Puedes ingresar la información de Proxy que corresponda al servicio externo que hayas elegido en el campo de “Configuración del proxy” ubicado bajo la opción de “Configuración” de este programa.
