Notes on Searching
 Your search requests are automatically sent to the pre-selected search engine you have chosen via the Search Settings screen. To use a search engine other than a pre-selected search engine, type in the Web address of your prefered search engine via the Web Address Input screen (), then proceed with your search.

 Please be aware that some Webpages that show up in the results of a search may not actually contain the keyword you used to run your search.

 With the exception of Websites run by Nintendo Corporation, Nintendo takes no responsibility for the content displayed via the search function.
