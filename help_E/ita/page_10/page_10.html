10      
  Domande e risposte
       
 
 
 
D
Perché non riesco a visualizzare alcune pagine web?
R
Il browser non può visualizzare i file in formato PDF e non è compatibile con alcuni tipi di file audio o video.
D
Il browser è in grado di riprodurre file Flash?
R
Il browser è in grado di riprodurre elementi compatibili con Flash 7. Gli elementi in formato Flash creati per una versione successiva potrebbero non essere visualizzati correttamente.
D
Perché alcuni siti web hanno tempi di caricamento particolarmente lunghi?
R
Se un sito web contiene molte immagini, il caricamento potrebbe essere più lento.
D
Posso aggiornare il mio blog e pubblicare messaggi nei forum con questo browser?
R
Sì, questo browser permette di fare entrambe le cose, tuttavia potrebbe non essere in grado di aprire determinate pagine web.
D
Posso impedire ad altri l'utilizzo del browser?
R
Sì, puoi usare la funzione di filtro famiglia della console per impedire ad altri utenti di usare il browser. Puoi anche usare un servizio per bloccare l'accesso di altri utenti a siti web pericolosi.
D
Che tipo di limitazioni è possibile impostare tramite il filtro famiglia?
R
Puoi impostare il browser in modo che richieda un PIN per l'accesso a Internet. Se utilizzi un servizio di filtro dei contenuti, ti verrà richiesto di inserire un PIN anche se desideri attivare o disattivare il blocco di pagine web dai contenuti ritenuti pericolosi.
D
Quante pagine web posso salvare nella mia lista dei preferiti?
R
Non puoi salvare più di 56 preferiti.
D
Posso ingrandire le dimensioni dei caratteri visualizzati a schermo?
R
Sì, puoi usare la funzione di zoom (premendo ) per leggere il testo più facilmente. In alternativa è anche possibile ingrandire i caratteri premendo  per attivare la modalità di visualizzazione su una colonna.
D
Posso usare il browser per effettuare download e upload di file?
R
Non è possibile effettuare download e upload di file usando questo browser.
D
Posso accedere alla mia casella di posta elettronica con questo browser?
R
Con questo browser è possibile avere accesso alla maggior parte dei provider di posta elettronica, ma non a tutti.
D
Posso usare il browser con il controller tradizionale, il Nunchuk™ o un controller per Nintendo GameCube™?
R
No, questi controller non sono supportati dal browser.
D
Come si eliminano i cookie?
R
Seleziona dapprima la voce Impostazioni nella pagina iniziale, poi Cancella i cookie.
D
È possibile eseguire ricerche con questo browser?
R
Sì. Per eseguire una ricerca seleziona l'icona Cerca: la puoi trovare nella pagina iniziale e nella barra degli strumenti.
D
Posso aggiornare la pagina web che sto visitando?
R
Sì, puoi aggiornare la pagina selezionando l'icona nella barra degli strumenti.
D
Posso visualizzare l'URL della pagina che sto visitando?
R
Sì, per visualizzare l'URL della pagina che stai visitando seleziona l'iconao l'iconasulla barra degli strumenti.
D
Posso vedere il certificato di sicurezza di una pagina web?
R
Sì, è sufficiente selezionare l'iconasulla barra degli strumenti.
D
C'è un limite massimo per i caratteri che posso inserire nel campo di immissione dell'URL?
R
Non è possibile inserire nel campo di immissione dell'URL più di 1024 caratteri.
D
Posso eseguire ricerche all'interno di una pagina web?
R
No, questo browser è sprovvisto di tale funzionalità.
D
Questo browser è provvisto della funzionalità di cronologia?
R
No, questo browser è sprovvisto di tale funzionalità.
D

R
Seleziona l'opzione Filtro famiglia nelle Opzioni Wii e segui le istruzioni che compaiono sullo schermo.
