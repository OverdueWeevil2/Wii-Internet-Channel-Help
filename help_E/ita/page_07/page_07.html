  Impostazioni
       
 
 
Tipo di zoom Sono disponibili due tipi di zoom.
Graduale Puoi decidere di quante volte ingrandire un'area della pagina.
Automatico L'area della pagina nella quale si trova il puntatore verrà ingrandita di un numero di volte predefinito.
Barra degli strumenti Seleziona la modalità di visualizzazione della barra degli strumenti.
Sempre attiva La barra degli strumenti verrà sempre visualizzata.
A scomparsa La barra degli strumenti verrà visualizzata automaticamente se punti il telecomando Wii verso il bordo inferiore dello schermo.
Manuale Visualizza la barra degli strumenti manualmente premendo .
Motore di ricerca Seleziona il motore di ricerca che vuoi usare per la funzione di ricerca del browser.
Google Ogni motore di ricerca opera in maniera diversa rispetto agli altri, quindi inserendo la stessa parola chiave in motori di ricerca diversi si otterranno risultati diversi.
Yahoo!
Cancella i cookie Cancella i cookie salvati.
Regola dimensioni schermo Modifica la larghezza dello schermo e aggiungi un margine di scorrimento verticale nell'area superiore e in quella inferiore dello schermo.
Impostazioni proxy Modifica le impostazioni relative all'uso di un server proxy.
Usa server proxy Utilizza un server proxy.
   Proxy Inserisci l'indirizzo del tuo server proxy o l'indirizzo IP.
   Porta Inserisci il numero della porta che comunica con il server proxy che stai usando.
Usa per HTTPS Utilizza un server proxy quando ti colleghi a una pagina sicura.
Autenticazione automatica Attiva l'autenticazione automatica per i server proxy che richiedono un'autenticazione.
   ID utente Inserisci un ID utente per l'autenticazione automatica.
   Password Inserisci una password per l'autenticazione automatica.
