10      
  Q&A
       
 
 
 
Q
Why won't this webpage display?
A
The browser can't display PDF files. Some types of movie and sound files are not compatible with the browser.
Q
Is the browser capable of displaying Flash files?
A
The browser can display Flash material that is compatible with Flash version 7. Flash material designed for later versions of the program may not display properly.
Q
Why does this website seem so slow to display?
A
Websites that use a large number of images may take longer to display within the browser.
Q
Can I update my blog and post on message boards with the browser?
A
Yes, the browser allows you to do both, but certain websites will not work with the browser.
Q
Can I control who can use the browser on my Wii?
A
Yes, you can use the Parental Controls feature to restrict who can use the browser. You can also use a content-blocking service to keep users from accessing harmful websites.
Q
What kind of restrictions can I put on the browser with the Parental Controls?
A
You can set the browser to require a PIN before it will allow a user to view websites. A PIN will also be required when you wish to activate or cancel the blocking of websites deemed to contain harmful content.
Q
How many Favorites can I have on my Wii?
A
You can save a maximum of 56 Favorites.
Q
Can I make the font any bigger?
A
Yes, you can zoom in on the text with  to make it easier to read. Alternatively, you can increase the text size by pressing  to switch the browser's display to single column mode.
Q
Does the browser allow me to download and upload files?
A
The browser doesn't allow you to download or upload files.
Q
Can I access my e-mail from the browser?
A
Most e-mail accounts work with the browser, but some may not.
Q
Are the Classic Controller, Nunchuk™ and Nintendo GameCube™ Controller compatible with the browser?
A
You can't use these controllers with the browser.
Q
How do I delete cookies?
A
Select the Settings icon on the Start Page, and then choose Delete Cookies.
Q
Is there a search function in the browser?
A
Yes. You can access the search function by selecting the Search icon located at the top of the Start Page and within the Toolbar.
Q
Can I reload the web page I'm viewing?
A
You can reload a web page by clicking on the Reload Page button in the Toolbar.
Q
How do I bring up the URL for a page I'm viewing?
A
You can display the URL for a page you are viewing by selecting either the Page Information button or the Security Information button in the Toolbar.
Q
How can I view a website's security certificate?
A
You can access this information by selecting the Security button () in the Toolbar.
Q
What is the longest possible address I can enter into the address field?
A
You can enter a maximum of 1,024 characters into the address field.
Q
Is there a way to search the contents of a single webpage?
A
The browser does not have this feature.
Q
Does the browser have a history feature?
A
The browser does not have this feature.
Q
I forgot my PIN for the browser. What do I do?
A
Select the Parental Controls option via the Wii Settings and follow the on-screen instructions.
