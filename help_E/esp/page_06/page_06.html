  Direcciones web
       
 
 
 
Cómo introducir la dirección de una página web

Apunta al icono y pulsa .

Aparecerá la pantalla del teclado.

 
 
 
  Escribe la dirección de una página web y confirma la entrada.

Puedes omitir "http://" al escribir una dirección web.

Por ej.: página oficial de Nintendo


     http://wii.nintendo-europe.com

            ↓

     wii.nintendo-europe.com

Para saber cómo usar el teclado de Wii, consulta la sección correspondiente en el manual de los canales y la configuración incluido con la consola Wii.
