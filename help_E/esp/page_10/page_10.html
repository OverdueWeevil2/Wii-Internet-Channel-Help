10      
  Preguntas y respuestas
       
 
 
 
P
La página web no se visualiza.
R
Este navegador no permite visualizar archivos PDF. Además, algunos formatos de archivos de vídeo y sonido no son compatibles con el navegador.
P
¿Puedo visualizar archivos con formato Flash en este navegador?
R
El navegador solo puede mostrar el material de Flash compatible con la versión 7 del programa. El material de Flash diseñado para una versión posterior del programa podría no visualizarse correctamente.
P
La página web tarda mucho tiempo en cargarse.
R
Los sitios web que usan un gran número de imágenes pueden tardar más tiempo en visualizarse.
P
¿Puedo actualizar mi blog y participar en foros con este navegador?
R
Sí, el navegador te permite realizar ambas cosas, pero algunos sitios web no son compatibles con este navegador.
P
¿Puedo controlar el acceso al navegador en mi consola Wii?
R
Sí, puedes usar la función de control parental para restringir el acceso al navegador. También puedes usar un servicio de bloqueo de contenidos para evitar que los usuarios accedan a sitios web potencialmente peligrosos.
P
¿Qué tipo de restricciones puedo imponer al navegador con el control parental?
R
Puedes configurar el acceso al navegador a través de una contraseña que el usuario deberá introducir para ver sitios web. También deberás introducir una contraseña para activar o cancelar el bloqueo de sitios web potencialmente peligrosos.
P
¿Cuántos favoritos puedo añadir en mi consola Wii?
R
Puedes guardar un máximo de 56 favoritos en tu lista.
P
¿Puedo ampliar el tamaño de la fuente?
R
Sí, puedes acercar el zoom con  para que el texto resulte más fácil de leer. También puedes aumentar el tamaño de la fuente cambiando la visualización de la página al modo de una columna con .
P
¿Puedo descargar o subir archivos con este navegador?
R
Este navegador no permite subir ni descargar archivos.
P
¿Puedo leer mi correo electrónico usando este navegador?
R
La mayoría de las cuentas de correo electrónico son compatibles con el navegador, pero es posible que haya algunas excepciones.
P
¿El mando clásico, el Nunchuk™ y el mando de Nintendo GameCube™ son compatibles con este navegador?
R
No puedes usar esos mandos con el navegador.
P
¿Cómo puedo borrar las cookies?
R
Selecciona el icono de configuración en la página de inicio y elige la opción BORRAR LAS COOKIES.
P
¿Hay una función de búsqueda en el navegador?
R
Sí, puedes acceder a dicha función eligiendo el icono de búsqueda situado en la parte superior de la página de inicio, así como en la barra de herramientas.
P
¿Puedo actualizar la carga de una página web que estoy viendo en ese momento?
R
Sí, basta con seleccionar el icono de actualización () en la barra de herramientas.
P
¿Cómo puedo ver la dirección exacta (URL) de la página web que estoy visitando?
R
Para ver la URL de la página web que estés visitando, selecciona el icono de información de la página () o el icono de información de seguridad () en la barra de herramientas.
P
¿Cómo puedo ver el certificado de seguridad de un sitio web?
R
Puedes acceder a esta información seleccionando el icono de información de seguridad en la barra de herramientas.
P
¿Cuál es la dirección más larga que se puede escribir?
R
Se puede introducir un máximo de 1024 caracteres en una dirección.
P
¿Hay alguna forma de realizar búsquedas dentro de una única página web?
R
Esa función no está incluida en este navegador.
P
¿El navegador cuenta con una función de historial de visitas?
R
Esa función no está incluida en este navegador.
P
He olvidado mi contraseña para el navegador. ¿Qué puedo hacer?
R
Elige la opción CONTROL PARENTAL en la configuración de la consola Wii y sigue las instrucciones que se mostrarán en pantalla.
Si tienes alguna pregunta o duda relativa al Canal Internet, visita la sección de soporte técnico de la página oficial de Nintendo (http://wiisupport.nintendo-europe.com) antes de buscar asistencia.
