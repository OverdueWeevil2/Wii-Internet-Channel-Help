  Configuración
       
 
 
Tipo de zoom Se pueden utilizar dos tipos de zoom.
Manual Ajusta el zoom de forma manual.
Automático Amplía automáticamente la zona a la que estés apuntando con el mando de Wii.
Visualización de la barra de herramientas Selecciona el modo de visualización de la barra de herramientas.
Permanente La barra de herramientas se mostrará permanentemente en pantalla.
Automática La barra de herramientas se mostrará de forma automática cuando saques el cursor por la parte inferior de la pantalla, y permanecerá visible mientras mantengas el cursor sobre ella.
Manual Muestra u oculta la barra de herramientas con .
Motor de búsqueda Elige un motor de búsqueda para realizar búsquedas desde el navegador.
Google Dado que cada motor de búsqueda funciona de forma diferente, a menudo la misma palabra clave puede tener distintos resultados.
Yahoo!
Borrar las cookies Elimina las cookies.
Ajuste de la pantalla Modifica la anchura de la pantalla, así como los márgenes superior e inferior.
Configuración del proxy Ajusta la configuración del servidor proxy.
Usar un servidor proxy Utiliza un servidor proxy.
   Proxy Introduce la dirección del servidor proxy o dirección IP.
   Puerto Introduce el puerto que comunica con el servidor proxy que estás usando.
Usar en
páginas seguras (HTTPS) 		Utiliza un servidor proxy al conectarte a una página web segura.
Autenticación automática Activa la autenticación automática de los servidores proxy que requieran autenticación.
   Usuario Introduce el nombre de usuario para los servidores proxy que se conectan automáticamente.
   Contraseña Introduce una contraseña para los servidores proxy que se conectan automáticamente.
