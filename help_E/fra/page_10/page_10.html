10      
  FAQ
       
 
 
 
Q
Pourquoi certaines pages ne s'affichent-elles pas?
R
Le navigateur ne peut pas afficher les documents au format PDF. De même, certains formats de fichiers vidéo ou musicaux ne sont pas compatibles.
Q
Le navigateur peut-il afficher les fichiers au format Flash?
R
Le navigateur peut afficher les fichiers Flash compatibles avec Flash version 7. Les fichiers destinés à des versions ultérieures du programme peuvent ne pas s'afficher correctement.
Q
Pourquoi certains sites s'affichent-ils lentement?
R
Certains sites qui affichent de nombreuses images peuvent nécessiter plus de temps pour s'afficher.
Q
Puis-je utiliser le navigateur pour mettre à jour mon blog ou poster sur un forum?
R
Oui, le navigateur vous permet de faire les deux, mais certains sites web ne sont pas compatibles avec le navigateur.
Q
Puis-je contrôler qui est autorisé à utiliser le navigateur sur ma Wii?
R
Oui, la fonction de contrôle parental permet de limiter l'utilisation du navigateur à certains utilisateurs. Vous pouvez également utiliser un service de blocage de contenu pour limiter l'accès à certains sites web.
Q
Quelles restrictions la fonction de contrôle parental permet-elle d'imposer pour la chaîne Internet?
R
Vous pouvez configurer le navigateur de telle sorte qu'un code soit nécessaire au démarrage. Ce code secret sera également demandé pour activer ou désactiver le blocage des sites web dont le contenu pourrait être dérangeant.
Q
Combien puis-je enregistrer de favoris dans ma liste?
R
La liste de favoris peut contenir au maximum 56 adresses.
Q
Puis-je augmenter la taille des caractères?
R
Oui, vous pouvez effectuer un zoom sur le texte en appuyant sur . Vous pouvez également augmenter la taille des caractères en passant en mode d'affichage vertical, en appuyant sur .
   
Q
Puis-je télécharger des fichiers?
R
Non, le navigateur ne permet pas de télécharger des fichiers.
Q
Puis-je accéder à mes e-mails depuis le navigateur?
R
La plupart des comptes e-mail sont compatibles avec le navigateur, mais certains ne le sont pas.
Q
Puis-je utiliser le navigateur avec la manette classique, le Nunchuk™ ou une manette Nintendo GameCube™?
R
Non, le navigateur ne peut pas être utilisé avec ces manettes.
Q
Comment puis-je effacer les cookies?
R
Sélectionnez l'icône des paramètres sur la page de démarrage, puis choisissez EFFACER LES COOKIES.
Q
Le navigateur dispose-t-il d'une fonction de recherche?
R
Oui. Sélectionnez l'icône de recherche en haut de la page de démarrage ou dans la barre d'outils.
Q
Puis-je recharger le site web que je suis en train de consulter?
R
Pour recharger un site web, sélectionnez RECHARGER LA PAGE () dans la barre d'outils.
Q
Comment puis-je afficher l'adresse (URL) de la page que je suis en train de consulter?
R
Sélectionnez INFORMATIONS SUR LA PAGE ( et ) dans la barre d'outils.
Q
Comment puis-je afficher le certificat de sécurité d'un site web?
R
Affichez ces informations en sélectionnant l'icône d'informations sur la page dans la barre d'outils.
Q
Quelle est la longueur maximale des adresses que je peux entrer dans le champ d'adresse?
R
Vous pouvez entrer au maximum 1024 caractères.
Q
Puis-je effectuer une recherche dans le contenu d'une seule page web?
R
Non, cette fonction n'est pas disponible dans le navigateur.
Q
L'historique des sites web visités est-il disponible?
R
Non, cette fonction n'est pas disponible dans le navigateur.
Q
J'ai oublié le code secret pour accéder au navigateur. Que puis-je faire?
R
Sélectionnez CONTROLE PARENTAL dans les paramètres Wii et suivez les instructions à l'écran.
