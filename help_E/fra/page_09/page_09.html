9      
  Glossaire
       
 
 
 
Adresse (URL)
Une adresse indique l'emplacement d'un site web sur Internet. On l'appelle aussi l'URL.

Navigateur
Un navigateur est un logiciel qui vous permet d'afficher les sites accessibles sur Internet. La chaîne Internet utilise un navigateur Internet complet développé par Opera Software.

Chercher
Effectuer une recherche sur Internet consiste à filtrer les informations disponibles sur le réseau selon certains mots-clés. La chaîne Internet vous permet d'utiliser à votre convenance les moteurs de recherche Google ou Yahoo!.

Proxy
C'est un serveur qui permet à l'utilisateur de se connecter indirectement à Internet en recevant en premier lieu les informations demandées avant de les transmettre à la machine de l'utilisateur.

Cookie
Un cookie est un ensemble de données qui contient des informations sur un site web que vous avez visité. Les cookies sont émis par ces sites web et sont stockés sur votre Wii.
